회원가입 불가능 기능
1) 101-101 표현식으로만 가능
2) 비밀번호와 비밀번호 확인과 일치시에 가능
3) 차량번호는 4자리 입력시에만 가능
4) 핸드폰번호 010-1111-1111 하이픈 사용후 가능

5) 회원가입시 자동으로 승인상태 D로 가입완료
6) 같은 아이디로 회원가입시
ㄴ> 이미 가입된 회원입니다. 혹은 관리자 승인 대기중입니다.

<로그인>
1) 승인상태가 A인 경우에만 로그인 가능
2) 첫 회원가입시 승인상태 D로 관리자가 승인시에만 로그인 가능
3) 등록된 아이디 없을시 로그인 불가능
4) 비밀번호 틀릴 시 로그인 불가능

<회원조회>
## 전체회원 조회
1) 아이디 조회시 한자리 숫자 로도 포함되는 아이디 전부 출력
2) 해당 내용 없을시 일치하는 아이디 없음을 알림

## 승인회원 조회(A: 승인 회원 D: 미승인 회원)
전체회원 조회의 1,2 같음
3) 미승인 회원만 전부출력
4) 승인할 회원 선택후 승인 버튼 클릭
5) 승인 후 승인된 회원포함 자동 재조회
6) 이미 승인된 회원 승인버튼 클릭시 이미 승인된 회원임을 알림

## 삭제회원 조회
전체회원 조회의 1,2 같음
3) 삭제할 회원 선택 후 삭제 버튼 클릭
4) OK 버튼 클릭시에만 삭제완료
5) 삭제 후 전체회원 자동 재조회

<게시판>
## 공지사항 
1) 제목 미입력시 등록 불가능
2) 내용 미입력시 등록 불가능
3) 등록 후 등록게시글 포함 전체 보여줌
4) 게시글 클릭시 해당 게시글 내용 보여줌
5) 게시글 클릭 후 수정가능
ㄴ> 이전 내용과 동일할 시 수정한 내용 없음을 알림
6) 게시글 클릭 후 삭제가능
ㄴ> 확인 및 취소 버튼으로 OK버튼 클릭시에만 삭제완료

## 1:1 게시판
1) 게시글 클릭 후 내용 확인 가능
2) 해당 내용에 답변시 공란시 등록 불가능
3) 답변 등록후 등록된 내용포함 자동 재조회
4) 답변 수정시 미수정시 수정된 내용없음을 알림
5) 수정 후 자동 재조회
6) 삭제할 게시글 클릭 후 삭제버튼 클릭 
ㄴ> 게시글 내용 전체 삭제됨

## 자유게시판
공지사항의 1,2,3 같음
4) 각 게시글 클릭시 해당 댓글 내용 두번째 테이블에 보여줌
5) 관리자도 댓글 작성 가능
6) 댓글 작성 후 전체 자동 재조회 
7) 댓글 더블 클릭시 삭제 가능
ㄴ> OK로 삭제함
ㄴ> 삭제 후 관리자에 의해 삭제되었음으로 흔적 남김
8) 게시글 수정가능, 미수정시 수정된 내용없음 알림
ㄴ> 마스터 기능으로 회원이 올린 게시글도 수정가능
9) 수정후 자동 재조회
10) 삭제할 게시글 클릭 후 삭제버튼 클릭 
ㄴ> 게시글 내용 전체 삭제됨
